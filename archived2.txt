What actions should i have for my screen writer ai?

----------------------- Q ----------------------
Great question. For a screenwriter AI, your set of custom actions will depend on the scope of tasks you want it to handle — everything from idea generation to formatting final scripts. Here's a breakdown of common and valuable actions you might want to expose through OpenAPI for a screenwriting assistant:

🔧 Core Actions for a Screenwriter AI
1. Generate Logline


Purpose: Generate a compelling one-sentence summary for a movie or episode idea.


Input: Genre, setting, main character, conflict.


Output: A short logline.


2. Outline Script


Purpose: Produce a scene-by-scene or act-based outline.


Input: Title, genre, summary/logline, structure (e.g., 3-act, 5-act).


Output: JSON or markdown outline.


3. Write Scene


Purpose: Generate properly formatted screenplay scene text.


Input: Scene description, characters, location, time of day, preceding scene.


Output: Screenplay scene in Fountain or Final Draft format.


4. Suggest Dialogue


Purpose: Help improve or generate character dialogue for a scene.


Input: Character name, scene context, tone/emotion, previous lines.


Output: Character dialogue lines.


5. Rewrite Scene


Purpose: Rewrite a given scene in a different tone or style.


Input: Original scene, desired tone (e.g., comedic, darker), length constraints.


Output: Rewritten scene.


6. Format Script


Purpose: Convert raw text to screenplay format (e.g., Fountain or Final Draft).


Input: Unformatted text or JSON structure.


Output: Formatted script text.


7. Character Bio Generator


Purpose: Create detailed character biographies.


Input: Character name, role in story, traits.


Output: Bio with appearance, backstory, goals, etc.


8. Conflict Brainstormer


Purpose: Suggest plot conflicts based on genre and theme.


Input: Genre, character archetypes, setting.


Output: List of potential conflicts or dilemmas.


9. Theme & Motif Suggestions


Purpose: Offer thematic or symbolic ideas aligned with the story.


Input: Genre, plot, character arcs.


Output: Themes, motifs, recurring visual/symbolic elements.



🧠 Bonus Advanced Actions (Optional)


Scene Emotional Map: Analyze emotional tone per beat or scene.


B-Story Generator: Create compelling subplots that support the main arc.


Script Coverage: Provide professional-style feedback on a script.


Genre Style Swap: Convert a scene or outline into a different genre.

Each of these tells the GPT how to handle the tool when it gets called.

🔧 Tool Instructions (Paste into each action's prompt box)
generateLogline

Generate a compelling one-sentence logline for a movie or episode based on the user's input. The logline should include the genre, setting, protagonist, and central conflict. Keep it concise, dramatic, and aligned with industry standards.


outlineScript

Create a script outline using the provided title, genre, logline, and structure format. Divide the outline into acts or scenes depending on the selected structure (e.g., 3-act, 5-act, nonlinear). Each section should summarize key events.


writeScene

Write a properly formatted screenplay scene using the provided description, characters, setting, and time of day. Use standard screenplay formatting and include action lines, dialogue, and scene headers. Keep tone consistent with prior context.


suggestDialogue

Generate character dialogue based on the provided context, character name, emotional tone, and previous lines. Keep the dialogue natural, engaging, and appropriate for the scene's situation and genre.


rewriteScene

Rewrite the given scene using the specified tone and maximum length. Preserve the original plot and characters but change the tone (e.g., make it darker, funnier, more suspenseful). Follow standard script formatting.


formatScript

Convert the provided raw screenplay text into proper screenplay format (such as Fountain). Apply standard formatting for scene headings, action, character names, and dialogue.


generateCharacterBio

Generate a detailed biography for the character using the provided name, role, and traits. Include backstory, motivations, relationships, and unique personality features. Use a tone consistent with the character’s story.


brainstormConflict

Suggest compelling internal and external plot conflicts based on the story's genre, setting, and character archetypes. Include both personal dilemmas and larger narrative threats or stakes.


suggestThemes

Propose meaningful themes and motifs based on the genre and plot summary. Themes should represent the deeper emotional and philosophical layers of the story. Motifs can be recurring visual or symbolic elements.


analyzeEmotions

Analyze the emotional tone and shifts in the provided scene. Return a list of emotions expressed throughout the scene, noting any major beats or emotional changes.


generateBStory

Create a complementary subplot (B-story) based on the main plot and list of characters. The B-story should support or contrast the main arc, provide depth, and tie into the characters’ development or theme.


scriptCoverage

Provide professional-style script coverage for the provided screenplay. Include strengths, weaknesses, and suggestions for improvement across structure, character, dialogue, and theme. Use a helpful and constructive tone.


genreSwapScene

Rewrite the scene in the specified new genre. Preserve the main beats of the original scene but transform the tone, style, and dialogue to fit the new genre (e.g., horror, sci-fi, romantic comedy, etc.).
